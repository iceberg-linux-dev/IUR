IUR (iceberg user repository)

WELCOME TO THE ICEBERG LINUX USER REPOSITORY
this repo is made to update your iceberg system via the package manager IPM
it also hosts the core of IPM for iceberg

now what is an IPM?
well IPM is an integrated package manager that is meant to preform like pacman but faster there's less hyphens and less words so you can get it done fast
Okay well why should i use iceberg if its just arch?
well thats where you are wrong its not arch iceberg is based on arch like many other distros but where iceberg is different is it is fast and is meant to be availible for everyone your hardware shouldnt define your capabilities and thats where iceberg comes in

you dont even have to activate the IPM its built in
heres all of the IPM commands and what they do
| Command | Action |
| :--- | :--- |
| `submerge <pkg>` | Install packages |
| `melt <pkg>` | Uninstall packages |
| `dive` | Update the entire system |
| `surface <query>` | Search for packages |
| `depth <pkg>` | View package details |
| `iceberg` | List installed packages |
| `freeze <pkg>` | Lock a package from updates |
| `thaw <pkg>` | Unlock a package |
| `update` | Update `ipm` from the IUR |
| `installer` | Launch the guided OS installer |

well how do i update?
glad you asked! IPM is made to update itself and its differnt from a system update normally as seen above a system update is dive but an IPM update is just update its easy simple and fast

it pulls updates straight from here and updates instantly
